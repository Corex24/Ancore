# Ancore Bot

A high-performance WhatsApp bot built with Node.js and Baileys.

## Setup
1. Clone the repository: `git clone https:                                      
2. Install dependencies: `npm install`
3. Create a `.env` file based on the `.env.example` file
4. Set up a PostgreSQL database and update the `DATABASE_URL` environment variable
5. Run the bot: `npm start`

## Features
* High-performance WhatsApp bot built with Node.js and Baileys
* Support for multiple commands and plugins
* PostgreSQL database integration
* Customizable sticker pack name and language
